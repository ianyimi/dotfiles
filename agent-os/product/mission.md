# Product Mission

## Problem

Setting up a new machine is time-consuming and error-prone. Maintaining consistent configurations across multiple machines is challenging. Managing secrets securely while keeping dotfiles in git requires careful coordination.

## Target Users

Personal use - for managing my own machines and development workflow.

## Solution

A complete automation system that combines:
- One-command bootstrap for fresh machine setup
- Secure secret management via self-hosted Bitwarden
- Configuration synchronization across all machines
- Project setup automation with git bare repositories
- AI-powered development standards via agent-os
